🎨 HTMLテンプレート集

HTMLエディターで使用できるサンプルテンプレートです。コピーして自由にカスタマイズしてください。

1. 基本的なWebページ

<!DOCTYPE html> <html lang="ja"> <head> <meta charset="UTF-8"> <title>私のWebページ</title> <style> body { font-family: Arial, sans-serif; margin: 40px; } h1 { color: #333; } </style> </head> <body> <h1>ようこそ！</h1> <p>これは私の最初のWebページです。</p> </body> </html>

2. カード型レイアウト

<!DOCTYPE html> <html lang="ja"> <head> <meta charset="UTF-8"> <title>カードレイアウト</title> <style> .card { background: white; padding: 20px; margin: 10px; border-radius: 8px; box-shadow: 0 4px 8px rgba(0,0,0,0.1); } .card-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); gap: 20px; } </style> </head> <body> <div class="card-grid"> <div class="card"> <h3>カード1</h3> <p>ここにコンテンツが入ります。</p> </div> <div class="card"> <h3>カード2</h3> <p>ここにコンテンツが入ります。</p> </div> </div> </body> </html>

3. フォーム付きページ

<!DOCTYPE html> <html lang="ja"> <head> <meta charset="UTF-8"> <title>お問い合わせフォーム</title> <style> form { max-width: 400px; margin: 0 auto; } input, textarea { width: 100%; padding: 10px; margin: 10px 0; } button { background: #007bff; color: white; padding: 12px 24px; border: none; } </style> </head> <body> <h1>お問い合わせ</h1> <form> <input type="text" placeholder="お名前" required> <input type="email" placeholder="メールアドレス" required> <textarea placeholder="メッセージ" rows="5"></textarea> <button type="submit">送信</button> </form> </body> </html>

🚀 使い方

上記のテンプレートをHTMLエディターにコピーして、プレビューで確認してみましょう！

  • モダンCSS
